# Security Policy

## Branch Protection
- Direct pushes to `main` are not allowed
- All changes require a Pull Request
- At least 1 approval required before merge
- Force pushes are blocked
- Admin bypass available for solo lab work

## What's Safe to Share
- pyATS test scripts
- Validation logic
- Test structure

## What's NOT Here
- Real device credentials
- testbed.yml (gitignored — contains IPs and passwords)
